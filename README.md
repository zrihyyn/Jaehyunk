# Jaehyun Kim Portfolio — minimal image gallery

## 사용법

1. `images` 폴더를 만듭니다.
2. 이미지 파일을 `work-01.jpg`, `work-02.jpg` 식으로 넣습니다.
3. `index.html` 안의 각 `<img>` 경로와 `width`, `height`를 실제 이미지에 맞게 수정합니다.
4. 각 `<li>`의 `--placeholder` 값을 원하는 로딩 배경색으로 바꿉니다.
5. 이름과 이메일 주소를 수정합니다.

## 구조

- 이미지 클릭 기능 없음
- 캡션/description/lightbox/metadata 없음
- 원본 비율 유지
- CSS columns 기반 masonry 형태
- 이미지별 로딩 배경색
- 모바일 반응형
